
# Sample K8s Deployment

Experimenting full-stack applications deployment on K8s




## Deployment

To deploy this project run

```bash
  kubectl -f infra/k8s --recursive
```


## Features

- K8s core (deployment, service, ingress, config-maps, secrets etc.)
- Postgres
- Frontend & Backend
